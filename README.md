# Advent of Code - 2025

My solutions to Advent of Code - 2025, written in node.

## Instructions

Solutions are organized in folders by days. To run a solution, execute the following at the root:

```
node index.js --day [day] <--test>
```

If you use the optional `--test` flag, the solution will be executed using the test input. Otherwise, the main input will be used.
